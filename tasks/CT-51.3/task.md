# CT-51.3: Add Line button on home screen

**Epic:** CT-51
**Depends on:** none

## Description

The user wants to reach the Add Line screen directly from the home screen, without having to navigate through the Repertoire Manager first.

## Acceptance Criteria

- [ ] A dedicated "Add Line" button is visible on the home screen for the active repertoire.
- [ ] Tapping it navigates directly to the Add Line screen.
- [ ] The Add Line screen remains accessible from the Repertoire Manager as before.
- [ ] The button is only shown when a repertoire exists (same condition as the other three action buttons).

## Notes

See updated spec in features/home-screen.md (Buttons section and Navigation Targets section).
