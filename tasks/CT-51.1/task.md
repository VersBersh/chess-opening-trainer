# CT-51.1: App bar colour and title size

**Epic:** CT-51
**Depends on:** none

## Description

The top app bar / banner on all screens currently uses a visually distinct background colour. The user wants it to match the screen background colour, and the title text to be smaller.

## Acceptance Criteria

- [ ] The top app bar background colour matches the screen background colour (transparent or `surface` colour token) on all screens.
- [ ] The title text in the top app bar uses a smaller text style (e.g. `titleMedium` or `titleSmall`).
- [ ] No other layout or functionality is affected.

## Notes

See updated spec in design/ui-guidelines.md (App Bar / Banner section).
