# CT-53: Test Notes

## Test files modified

- `src/test/widgets/move_pills_widget_test.dart`

## What each test covers

### Modified tests

1. **"labeled and unlabeled pills have identical fixed height"** (line 233)
   - Changed expected height from `50.0` to `46.0` to match the new pill height (`_kPillMinTapTarget` 32 + `_kLabelSlotHeight` 14 = 46).
   - Updated inline comment to reflect the new constant values (32 instead of 36, 46 instead of 50).

2. **"each pill tap target is at least 32 dp tall"** (was "at least 36 dp tall")
   - Renamed test to reflect the reduced tap target.
   - Changed assertion from `greaterThanOrEqualTo(36)` to `greaterThanOrEqualTo(32)`.

### New test

3. **"board-to-first-row gap equals inter-row pill-body gap"**
   - Renders `MovePillsWidget` in a 150 dp wide container with 4 pills, forcing 2 wrapped rows (2 pills per row).
   - Measures the board-to-first-pill-body gap: distance from the pill area `Padding` top edge to the top of the first `GestureDetector` (pill body).
   - Measures the inter-row pill-body gap: distance from the bottom of row 0's `GestureDetector` to the top of row 1's `GestureDetector`.
   - Asserts the two gaps are equal, enforcing the uniform-gap invariant from the acceptance criteria.

### Existing test verified unchanged

4. **"label slot does not cause wrapped rows to overlap"**
   - No changes needed. The `row0Rect.bottom <= row1Rect.top` assertion is dimension-independent and will continue to pass with the new pill height and spacing.
