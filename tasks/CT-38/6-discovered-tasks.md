# CT-38: Discovered Tasks

## 1. Auto-open label editor after "Add name" dialog action

- **Suggested task ID:** CT-39
- **Title:** Auto-open label editor when user chooses "Add name" from no-name warning
- **Description:** When the user taps "Add name" in the no-name warning dialog, they currently return to the screen and must manually tap a saved pill to open the inline label editor. Instead, automatically open the label editor on the last saved pill so the user can immediately start typing a name.
- **Why discovered:** During CT-38 implementation, noted as a UX improvement in the plan's open questions (Risk #4). The current flow requires extra taps to actually add a name after choosing "Add name".
