## Development

### Debug window size

In debug builds (`flutter run -d windows`), the window opens at Pixel 5 phone
dimensions (393x851) so you can preview mobile layout on desktop. The window is
fully resizable — the phone size is just the default.

Release builds use the standard 1280x720 default.

To change the debug dimensions, edit the `#ifdef _DEBUG` block in
`windows/runner/main.cpp`.
